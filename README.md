## Introducción
Este proyecto sera desarrollado con el propósito de gestionar y organizar el control de dotaciones dentro del instituto colombiano de bienestar familiar como operador tendremos a la asociacion mundos hermanos El sistema permite administrar diferentes tipos de dotaciones para los niños niñas y adolecentes, esto incluye dotaciones de ropa, entrega de dinero, auxilios de pasajes y suministros de aseo y accesorios personales como juego de sabanas facilitando el registro, seguimiento y control de cada recurso entregado.

La aplicación busca optimizar los procesos administrativos, reducir errores en la gestión manual y mantener un historial claro y ordenado de las dotaciones realizadas. Además, permite centralizar la información para mejorar la eficiencia, la transparencia y el control de inventarios y entregas.

###  Características

- registro de jovenes <br>
- gestion de dotaciones <br>
- control de inventario<br>
- historial de entregas <br>
- reportes y consultas<br>
