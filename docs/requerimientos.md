# NRF
### RNF01. Seguridad
El sistema deberá garantizar la protección de la información mediante autenticación de usuarios y contraseñas seguras con caracteres especiales.

### RNF02. Rendimiento
El sistema deberá responder a las solicitudes de los usuarios en un tiempo máximo de 3 segundos en operaciones comunes.

### RNF03. Disponibilidad
La plataforma deberá estar disponible para los usuarios la mayor parte del tiempo, garantizando continuidad en el servicio.

### RNF04. Usabilidad
El sistema deberá contar con una interfaz amigable, intuitiva y fácil de usar para administradores y usuarios.

### RNF05. Escalabilidad
El sistema deberá permitir futuras ampliaciones, incluyendo nuevos módulos o tipos de dotaciones.
