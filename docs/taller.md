# Taller - Métodos HTTP y Códigos de Estado

Identificar los principales métodos HTTP y los códigos de respuesta utilizados en servidores y APIs REST.

## Métodos HTTP

Complete la tabla consultando la función de cada método.

| Método | Función | Ejemplo |
| ------ | ------- | ------- |
| GET    |Obtener  |Get list user|
| POST   |Enviar   |Post new user|
| PUT    |Actualizar    |Put date the user|
| DELETE |Eliminar      |Delete user data base|

## Códigos HTTP

| Código | Nombre               | Significado |
| ------ | -------------------- | ----------- |
| 200    |OK                    |Exito en la solicitud de un servidor Web|
| 201    |Created               |Solicitud HTTP ha tenido exito          |
| 400    |Bad Request           |Servidor no procesa la solicitud             |
| 401    |Unauthorized(No autorizado)          |Solicitud no completada, carece de credenciales de acceso|
| 403    |Forbidden             |El servidor entiende la solicitud, pero el usuario no tiene permisos|
| 404    |Not Found(No encontrado)             |El servidor no puede localizar la pagina             |
| 500    |Internal server error                |Fallo interno del servidor impidiendo respuesta      |
